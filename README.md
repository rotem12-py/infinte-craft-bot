# Infinite Craft Bot Explorer

*the autocrafter - to infinity and beyond!*

This simple python program uses the [infinitecraft module](https://pypi.org/project/infinite-craft/) to automate the game and interface with it



## What each file in the project does

- `main.py`: this is the main and primary file in the project, its the bot itself which is responsible for getting items.
  generates: `checked.txt` and `discoveries.json`.
- `checked.txt`: a text file which stores element already exhausted so when restarted the bot wont go over them again
- `discoveries.json`: stores your elements in a json file, automatically created by the main script.
- `generator.py`: a simple, short script that allows users to create custom elements and add them to the game.
- `conv.py`: this file takes the json file, which the web game cant use and understand, and turns it into output.ic(see below), a file formatted in a way the game can read.
 generates: `output.ic`.
- `output.ic`: this is the result of the conv script, its a file with your elements stored in it which you can upload to the game and play with what you got.

## Installation

this guide assumes [python and pip](https://www.python.org/downloads/) already installed and functional. if not, download them

to install and run this program, you need to follow 3 simple steps.

1. Either clone this repo with `https://github.com/rotem12-py/infinte-craft-bot.git` or install the zip directly and extract it.
2. You need to install all required packages with `pip install -r  .\requirements.txt`, after that the project is ready and you can run it!
